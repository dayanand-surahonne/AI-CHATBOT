# 🤖 AI Chatbot (Full-Stack Web Application)

## 📌 Overview

This project is a full-stack AI chatbot web application that generates intelligent responses using the OpenAI API.
It provides a smooth and interactive chat experience with a clean and responsive user interface.

---

## ✨ Features


* 💬 Real-time chatbot interaction
* ⚡ Fast and responsive UI
* 📱 Mobile-friendly design
* 🔄 Dynamic message rendering
* 🔗 OpenAI API integration
* ❗ Error handling and validation
* 🔐 Secure API key management using environment variables

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express.js
* **API:** OpenAI API
* **Tools:** Git, Postman

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone https://github.com/dayanand-surahonne/AI-CHATBOT.git

### 2️⃣ Navigate to project directory

cd AI-CHATBOT

### 3️⃣ Install dependencies

npm install

### 4️⃣ Create a .env file and add your OpenAI API key

OPENAI_API_KEY=your_api_key_here

### 5️⃣ Start the server

npm start

## 🚀 How It Works

* User enters a message in the chat interface
* Request is sent to the backend (Node.js + Express)
* Backend calls OpenAI API
* AI-generated response is sent back and displayed instantly
